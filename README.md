# Currency Converter

A modern currency converter application built with React that allows users to convert currencies using real-time exchange rate data. This project was created to strengthen my understanding of API integration, state management, component-based architecture, and responsive user interface development.

## Overview

This application enables users to quickly convert amounts between different currencies using current exchange rates. The project focuses on delivering a clean user experience, efficient data handling, and responsive design across devices.

## Features

* Real-time currency conversion
* Dynamic exchange rate fetching through API integration
* Fast and responsive user interface
* Currency selection dropdowns
* Instant conversion updates
* Mobile-friendly design
* Error handling for API requests
* Clean and reusable React components

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd currency-converter
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
